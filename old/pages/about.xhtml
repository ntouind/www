社團簡介
前言

　IND全名為INSTITUTE OF NETWORK DEVELOPMENT，中文叫網路發展學會。是由海大電算中心老師與一群熱心的學生組成的團體，如同其他學校的網路策進會一樣，主要目的為促進學校網路的發展，提供本校師生網路服務。IND現在也轉型為學術性社團。

　IND最主要的工作就提供全校師生網路服務、伺服器維護以及全校性教學活動。目前所提供的網路服務項目有BBS、FTP、網頁與電子信箱等等，未來將會有電子相簿等更多元化的服務。但是宿舍網路與校園無線網路並非IND的服務範疇。

　IND所管理的Server多為UNIX系統，尤其是以FreeBSD為主，所以IND成員在這裡學習到與MS Windows完全不同的作業環境，能夠學習各種Server的架設管理，更可以藉由全校教學活動中擔任講師的機會，進而訓練自己的口才、台風等等。另外也能學到電腦組裝、網頁編寫、PHOTOSHOP實務等各式各樣的電腦技能。總歸一句話，只要是能和電腦沾上邊的事物都可以在這裡學習到。

 

發展延革
1994.12
時電研一的 shopping (蕭志平)帶著電三 bob (名字忘了), 船二ylchang (張毓麟) 與電一 xcape (洪錦森), 開始計中的另類網路服務. 計中陳傳宗老師爭取一 working room 供機器擺放使用.
   
1995.2
使用當時計中最新機種 486 dx2-66 兩部跑起 Linux, 以線上申請方式提供大一至大四 email 帳號服務. 一部名為 happy, 另一部命為 fun.
   
1995.夏
因需求量高過預期, 開始有組織團體的念頭Working Room 搬家, 由計中四樓 HP720 room 搬至三樓 MAC 教室四處招募人才加入, 夜航管三 tjm (蔡忠明)等正式加入.
   
1995.9
開會決定未來服務方向為協助計中管理 email 及 www server 等服務. 確定成立一非正式社團組織, 名為 "網路發展學會" (還是協會!?), 英文為 Institute of Network Developement, 簡稱 IND. 第零屆會長由 xcape 擔任.
   
1995.10
拿到 Intel Zappa Pentium 主機板, 正式建立 IND server. 以單顆 Pentium-90 加上 Linux, 開始正式將全校帳號建入, 提供 Email, WWW 等各式服務. 另建起 joy.ntou.edu.tw 做為海大 WWW server 使用.
   
95'~96'.冬
正式對外提供網路服務課程, Email 使用, UNIX 環境認識等
   
1996.2
經陳老師大力協助, 計中請購購入 DEC Alpha Server 1000, 單顆 DEC 233Mhz CPU, 64MB RAM, 2.4 GB HDD, 準備取代現有之 PC Linux.
   
1996.4
DEC Alpha Server 正式取代 PC Linux, 成為 IND 正式 Server 機種
   
1996.夏
電二 dada (饒仁達), bigbird (林宗賢)等生力軍正式加入 IND 行列 系統由 bigbird, dada, xcape 三人負責, ylchang 接掌造船系所管理工作. 選出 dada 擔任第一屆會長.
   
2008.3
改組為學術性社團
