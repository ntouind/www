組織章程
第一章 總則
  • 本會定名為國立台灣海洋大學網路發展協會(Institute of Network Development)，簡稱IND，以下簡稱本會。
  • 本會會址設於基隆市中正區北寧路2號。
第二章 宗旨
  • 本會之成立宗旨
    1. 促進海洋大學之網路發展。
    2. 協助電子計算機中心管理網路事宜。
    3. 研究各種網路技術及電腦相關知識。
    4. 促進海洋大學成為基隆市網路中心，並協助基隆市的網路發展。
第三章 一般會員
  • 凡擁有國立台灣海洋大學之學生身分，向IND線上會員系統登記者即成為一般會員。
  • 本會會員應享之權利：
    1. 使用會內各項資源，如電腦、圖書、印表機等。
    2. 參與各項技術討論。
    3. 參與IND舉辦之課程與活動
    4. 享有發言權，對各項會務有意見者，得隨時提出反應。
    5. 享有選舉權、被選舉權及罷免權。
  • 本會會員應盡之義務：
    1. 遵守會員大會通過之章程及各項規定。
    2. 積極參與會內舉辦之各項活動。
    3. 研究電腦相關技術, 並且發表專題。
    4. 貢獻心力予學校，協助校園網路之發展。
    5. 根據每學期之經費規劃繳交社費。
    6. 因為不定期之活動或是添購設備所產生之臨時社費需要給予支援。
  • 若以下事件發生，則將永久凍結該會員之所有權利與義務：
    1. 干擾本會運作秩序。
    2. 干擾活動屢勸不聽者。
    3. 重大過失。
    4. 被其他會員投票開除，須與會半數同意。
第四章 核心會員
  • 擁有一般會員資格，對IND具有重大貢獻者（例如但不限於擔任IND幹部，接管重要伺服器，或是擔任長期講師），可向會長主動提出。
  • 本會幹部與各組負責人與Server組公眾伺服器管理者自然擁有核心會員資格。
  • 會長須於每次期初大會之前檢視所有成員, 審核所有核心會員之申請, 並且於期初大會公佈該學期之核心會員名單. 若有任何異議直接於大會表決處理。
  • 若某成員於每學期初之審查發現不再符合核心會員資格，則恢復為一般社員身分。
  • 核心會員應享之權利：
    1. 一般會員之所有權利。
    2. 優先享用IND提供之各式資源。
    3. 對本會核心運作擁有先行裁決權。(參見會員大會章)。
  • 核心會員應盡之義務：
    1. 一般會員之所有義務。
    2. 監督IND之運作。
    3. 貢獻心力於IND之重要事務運作上。
  • 若是有重大過失, 比照一般會員辦理。
第五章 會員大會
  • 會員大會為本會之監察暨最高決策機構，由會長召集並擔任主席，每學期召開期初及期末會員大會，討論各項有關會務, 決定該會期IND各組 之主要工作方向。
  • 會員大會不出席之會員視同放棄權利，並且應遵守會員大會之決議，不得提出異議。
  • 必要時會長經由幹部會議同意後，或經由會員五分之一連署提議後，得召開臨時會員大會，決定會內重要事務。
  • 本會之重大決議需經大會全體會員二分之一以上出席，多數表決通過後方能執行。而若是全體會員出席率低於二分之一，而核心會員出席率達總核心會員數二分之一以上，則出席之核心會員之多數決議結果擁有相同效力。
  • 會員大會之權力如下：
    1. 選舉罷免會長、幹部任命之同意權。
    2. 經費收支與帳目之審核。
    3. 重大決議案之同意及審核。
    4. 通過修訂本章程。
  • 大會對預算、決算、行事曆，行使同意權時，採絕對多數決議之。
第六章 組織圖
第七章 會長 副會長
  • 會長為本會最高領導人、負責人。
  • 會長與副會長由全體會員選出。
  • 會長之任期以一學年為一任，不得連選連任。
  • 會長之職責如下：
    1. 對內總理、統轄會務，並對外代表本會。
    2. 對本會合作組織負責(學校, 計算機中心)。
    3. 大會召開前，須邀請相關單位列席。
    4. 批示各項企劃案。
    5. 聽取顧問團與導師的意見。
    6. 每個月初向全體成員報告社團運作現況。
    7. 其他依本章程賦予之。
  • 副會長之職權如下：
    1. 協助會長處理本會各類事務。
    2. 領導行政部門, 負責各組工作的督導。
    3. 協助各部門的溝通與合作。
  • 會長因故無法視事時，由副會長代行職權。會長、副會長均無法視事時，由行政部推派代表代行其職權。
  • 會長缺位時，由副會長代之但須於四週內通告全體會員並補選會長其任期內以補足原任會長未滿任期為止。
  • 如會長於任期結束前四週缺位時，由副會長代理至學期結束。
第八章 文書 總務
  • 文書及總務為本會既定之幹部，與會長、副會長選舉當天，由全體會員選出，任期等同於會長、副會長，連選得連任。
  • 文書的職權如下：
    1. 會內各項會議記錄之整理與保存。
    2. 本會各項文件及教學講議之整理保存與建檔。
    3. 協助其他各組之書面文件。
    4. 負責本會各項公文之傳送及接收。
    5. 維護IND文件系統。
  • 總務的職權如下：
    1. 經理本會之財產管理清點、會費收支、預算編列等事務。
    2. 負責每次活動費用之結算。
    3. 定期於會員大會報告本會財務狀況。
    4. 保管票據以供查對。
    5. 向課輔組與計中申請可申請之經費。
    6. 管理IND之練習機的租借與收回。
    7. 管理IND之網路(IP)使用權與機器擺放使用權之租借與收回。
    8. 需要外借設備時代表IND申請(例如計中與課輔組)。
第九章 行政部
  • 行政部設有活動組、教學組、美宣組、server組、與Meeting組。可視需要增減組別。
  • 行政部應協助各級幹部處理各項事務。
  • 各組置組長一人，各組人員皆由會長任命之，任期皆與會長同。
  • 本會各組之職責如下：
    • 教學組：
      1. 全校性基礎資訊教育的推廣。
      2. 每學期之教學課程的規劃。
      3. 寒暑訓教學之規劃與舉辦。
      4. 研究電腦軟硬體及最新之發展。
      5. 培養成員基本之電腦操作能力。
      6. 培養組員之教學能力。
      7. 培育選擇伺服器接任人選提供給伺服器組。
      8. 將教學文件整理並且公佈於IND文件系統。
    • 伺服器組：
      1. 研究各種伺服器之架設及其相關之技術。
      2. 協助本會管理各項伺服器。
      3. 與教學組合作, 提供伺服器相關之教學內容。
      4. 帶領適合之繼任者熟悉伺服器管理, 並在畢業前交接。
      5. 整理教學與管理文件並且公佈於IND文件系統。
    • 活動組：
      1. 籌劃、設計、辦理各學期之活動。
      2. 社團博覽會與社團評鑑。
      3. 負責本會會員聯誼性之活動。
      4. 負責對外之聯誼工作。
      5. 負責尋找舉辦可以提昇IND名聲的活動。
      6. 招募新生。
    • 美宣組：
      1. 各項活動之宣傳海報。
      2. 會場佈置之工作。
      3. 美術相關的外包案。
      4. 管理並隨時更新各宣傳通道。
    • Meeting組：
      1. 學期初開會分配每個會員不同之研究專題。
      2. 定期舉辦Meeting以彼此交流。
      3. 追蹤各專題之進度。
  • 各組其他職責由主席依本章程賦予之。
第十章 導師
  • 為了引導IND之正常運作，以及與計算機中心之有效溝通，計算機中心網路組組長為本會之當然指導老師，對IND之運作提出建言與監督。同時視需要延聘額外指導老師處理會務。
  • 會長有義務定期向導師做會內運作之相關報告。
  • 導師可以對社長與IND成員提供建議，但不做強制干預。
第十一章 顧問團
  • 顧問團之成員，由在學之行政幹部任命之。
  • 顧問團得對新上任之會長，提出幹部名單之建言。
  • 其職責在指導及協助新上任幹部，籌畫及協辦幹部訓練，監督本會行政運作。
  • 顧問團不參與本會事務之決定，其建言宜以書面行之。
第十二章 選舉
  • 本會會長選舉，採普通、平等、直接、無記名方法遴選會長，任期一年，不得連選連任。
  • 凡本會會員，除非不得已情況下，二年級始得登記競選會長。
  • 選舉時應邀請指導老師列席指導。
  • 新任會長、副會長於選舉結果確認後職權立即生效。
第十三章 罷免
  • 會長若因犯重大過失或未盡職責時，凡連署會員五分之一以上得提案罷免會長。
  • 罷免案需全體會員二之一以上出席，出席會員三之二通過後始成立。
  • 大會通過罷免案後，需於一個月內重新選舉。
  • 重新選舉期間，由副會長代理會長職務。
  • 新任會長選出後，前副會長需率領行政部下各級幹部總辭，並由新任會長所提出的幹部名單就任其缺。
第十四章 附則
  • 本章程所稱之法為：大會通過會長公佈之法規。
  • 各法與本章程牴觸者無效。
  • 本章程之增修，須於二之一以上會員出席之大會提出後，交付大會表決,凡出席會員三分之二以上同意，本章程修改之。
  • 本章程於會員大會通過後，公佈施行之。
  • 經費之使用應會長及總務事先同意，並且簽證、收據及發票應留存，在會員大會上公布使用情形及結餘，若有貪污舞弊之情形依法規嚴懲。
  • 本會各項會議程序，悉依會議規範之規定進行。
